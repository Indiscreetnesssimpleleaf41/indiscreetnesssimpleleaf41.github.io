## 📖 Using markdown for your notes

Markdown uses simple characters to format text without complex toolbars. Use these basic patterns to start:

- Use one # for a main title, ## for a section, and ### for a subsection.
- Use one asterisk around a word to italicize it.
- Use two asterisks around a word to make it bold.
- Put a dash and a space before a line to create a bullet point.
- Use square brackets for the text and parentheses for the link to insert a URL.

This method keeps your documents clean and readable as plain text files while allowing for professional formatting when you export them.
